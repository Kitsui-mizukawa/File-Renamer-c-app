# File Renamer – C++ Desktop Utility

A lightweight Windows desktop application built with C++ for fast and efficient batch file renaming. Dirancang untuk pengguna yang membutuhkan proses penamaan ulang file secara cepat, konsisten, dan bebas kesalahan.

# Fitur Utama

Batch Rename — Mengubah nama banyak file sekaligus dalam satu klik.

Custom Prefix & Suffix — Tambahkan pola nama sesuai kebutuhan.

Auto Numbering — Menambahkan penomoran otomatis untuk menjaga urutan file.

Simple & Clean UI — Antarmuka minimalis yang mudah dipahami.

Fast Processing — Dibangun dengan C++ untuk performa tinggi.

# Bahasa

C++ (92%)

C (8%)

Menggunakan WinAPI dasar untuk antarmuka dan operasi file.

# Tujuan Aplikasi

Aplikasi ini dibuat untuk mempermudah proses penamaan ulang file dalam jumlah besar—cocok untuk pengelolaan dokumen, foto, dataset, atau file proyek yang membutuhkan konsistensi penamaan.
